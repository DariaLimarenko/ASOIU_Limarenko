## Ответы на контрольные вопросы

### 1. Что такое расстояние Левенштейна?

Расстояние Левенштейна (редакционное расстояние) — это минимальное количество операций редактирования, необходимых для преобразования одной строки в другую. К таким операциям относятся удаление символа (delete), добавление символа (insert) и замена одного символа другим (replace). Расстояние определяется как минимальное число действий, которое требуется для преобразования первой строки во вторую.

### 2. Что такое расстояние Дамерау-Левенштейна?

Расстояние Дамерау-Левенштейна является модификацией расстояния Левенштейна. В дополнение к операциям вставки, удаления и замены символов учитывается ещё одна операция — транспозиция (transposition), то есть перестановка двух соседних символов. Такая операция считается за одно действие. Использование расстояния Дамерау-Левенштейна позволяет более точно находить слова при наличии опечаток, возникающих при вводе текста.

### 3. Объясните алгоритм Вагнера-Фишера для вычисления расстояния Дамерау-Левенштейна.

Алгоритм Вагнера-Фишера используется для вычисления расстояния между двумя строками на основе динамического программирования. Для этого создаётся матрица размером (M+1) × (N+1), где M и N — длины сравниваемых строк. Первая строка и первый столбец матрицы инициализируются значениями, соответствующими количеству операций вставки и удаления символов. Далее элементы матрицы вычисляются последовательно: для каждой ячейки выбирается минимальное значение из операций вставки, удаления и замены символа. При вычислении расстояния Дамерау-Левенштейна дополнительно проверяется возможность транспозиции соседних символов. Значение правого нижнего элемента матрицы является искомым расстоянием между строками.
